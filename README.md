# odin-landing-page

This is a standard landing page that was created to match the provided full design mockup that was given, along with the provided colors and fonts. It was created using HTML and CSS, so that I could apply the knowledge I've gained from The Odin Project thus far.

As this was the first time using Flexbox, that was the most challenging aspect of this project by far, but thanks to all of the additional resources to learn from beforehand, this project was really nice.

As for any potential future features, making this design mobile-friendly, creating additional pages for the navigation to link to, and adding interactivity via JavaScript would all be very nice.
